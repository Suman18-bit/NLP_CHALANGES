## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Suman18-bit/NLP_CHALANGES.git
cd NLP_CHALANGES
```

### 2️⃣ Install All Dependencies

```bash
pip install nltk spacy gensim transformers torch \
            scikit-learn pandas numpy matplotlib \
            vaderSentiment wordcloud
```

### 3️⃣ Download spaCy Language Model

```bash
python -m spacy download en_core_web_sm
```

### 4️⃣ Launch Jupyter

```bash
jupyter notebook
```

> 💡 **Prefer Google Colab?** Upload any `.ipynb` directly to [colab.research.google.com](https://colab.research.google.com) — no local setup needed!
